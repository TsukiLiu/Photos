# 买家秀 Skill

这个文件夹是一个本地 Codex skill，用来生成电商 AI 买家秀提示词。它会把商品图作为唯一服装来源，把参考图只当作姿势、场景、镜头、光线和生活感参考，输出可直接复制到 Lovart、TapNow、即梦等工具里的中文 `正向关键词` / `负面关键词`。

## 适合的请求

- 图1上衣，图2裤子，图3买家秀参考风格
- 图1外套，图2内搭，图3裤子，图4场景参考
- 男性模特 / 女性模特
- 衣服还原真实
- 背景不要照抄参考图
- 模特姿势不要复刻参考图
- 图片不要有滤镜感
- 背景不要有路人

## 强制规则

- 所有生成图都要求全身展示：竖版3:4，头顶到鞋子完整入镜。
- 服装只以商品图为准，参考图不能改变衣服、裤子、内搭或配饰。
- 图片必须自然真实，无强滤镜、无调色感、无棚拍大片感。
- 模特必须是亚洲男性或亚洲女性，场景必须是国内城市或国内生活空间。
- 不使用日韩、日系、韩系、欧美街拍等方向描述。
- 不照抄参考图中的模特脸、发型、表情、穿搭、姿势和原背景。
- 参考图背景只能提供氛围，最终背景要重新生成。
- 背景不能有路人、人群、游客、骑车人或虚化人影。
- 模特动作要自然丰富，避免直立死板站姿。

## 建议输入格式

```text
图1外套，图2内搭，图3裤子，图4场景参考，男性模特
```

也可以追加修补要求：

```text
裤子生成错了，背景不要照抄，动作丰富一点
```

## 输出格式

```text
正向关键词：
...

负面关键词：
...
```

## GitHub 上传

上传到 GitHub 时，保留这个文件夹结构即可：

```text
买家秀/
  SKILL.md
  readme.md
```

后续在支持 Codex skills 的环境里安装或同步该文件夹后，即可通过“买家秀”这个 skill 名称调用。
