# 买家秀提示词生成

## 使用场景

当用户需要根据商品图和参考图生成 AI 买家秀描述词时使用本 skill，典型请求包括：

- 生成买家秀正反描述词
- 图1上衣，图2裤子，图3参考风格
- 男性模特 / 女性模特
- 衣服还原真实
- 背景不要照抄参考图
- 模特姿势不要复刻参考图
- 去掉背景路人
- 图片不要有滤镜感

不要把参考图当作服装真相。商品图才是服装、面料、结构、颜色、Logo、口袋、裤型、鞋包等信息的唯一来源。

## 输出格式

默认输出一组可直接复制的中文提示词：

```text
正向关键词：
...

负面关键词：
...
```

如果用户要求多组姿势、男女分组、多个参考图或多个产品组合，按组分别输出，每组都必须有独立的 `正向关键词` 和 `负面关键词`。

## 强制要求

每次生成买家秀提示词都必须满足以下规则：

1. 全身展示
   - 必须写明 `竖版3:4，全身构图，头顶到鞋子完整入镜，保留脚下地面空间`。
   - 禁止半身图、七分身、裁掉鞋子、裁掉裤脚、头顶裁切。

2. 真实手机拍照感（核心风格）
   - **正向必须写明**：`iPhone原相机直出风格、不做后期调色、色彩真实不饱和不过曝、白天户外自然散射光带轻微阴影和轻微过曝、背景适度虚化但保留环境纹理、轻微手机镜头边缘畸变、自然噪点颗粒感`。
   - **正向必须写明**：`皮肤干净清爽带自然健康光泽但不磨皮过度、布料纹理真实可见、像朋友用手机在街上随手拍的生活照、有一点不完美但非常真实可信`。
   - 禁止强滤镜、复古调色、棚拍大片感、商业大片感、专业相机拍摄感、影棚灯光、柔光箱打光、CG感、3D感、AI感、AI重绘、AI生成图、AI绘图、AI人像、AI摄影、计算摄影痕迹、深度学习生成感、Stable Diffusion风格、Midjourney风格、生成式AI痕迹、模型生成感、机器生成感、不真实光影、塑料皮肤、磨皮过度、过度完美、过度平滑皮肤、皮肤像塑料假人、皮肤像蜡像、皮肤完全没有纹理、过度精致对称五官、对称脸、完美到不像真人、眼神呆滞、表情僵硬不自然、过度完美笑容。
   - 禁止皮肤出现明显瑕疵：负面必须加 `面部痘印、痘痘、痤疮、明显疤痕、皮肤粗糙、毛孔粗大、油光满面、肤色暗沉不均`。正向不要写 `真实皮肤带轻微毛孔和瑕疵`，避免生成令人不适的面部瑕疵。

3. 服装只看商品图
   - 使用 `图X作为唯一服装参考` 的表述。
   - 明确锁定颜色、面料、版型、领型、袖长、门襟、纽扣/拉链、口袋、拼接线、Logo/文字位置、裤腰、裤型、裤脚等。
   - 如果有内搭，必须单独锁定内搭结构和露出层次。
   - 如果生成结果出现面料漂移，要在正向里直接写正确面料，在负面里禁止错误面料。例如：`不是麂皮，不是磨砂皮，不是皮革，不是绒面`。

4. 模特和场景必须是国内语境
   - 使用 `亚洲男性模特` 或 `亚洲女性模特`，并写明 `中国城市日常买家秀`。
   - 不要出现 `日韩风`、`日系`、`韩系`、`欧美街拍` 等描述。
   - 场景写国内城市街区、国内现代公寓、国内咖啡店、国内生活方式店铺、国内社区步道等。

5. 不照抄参考图
   - 参考图只能提取大类氛围：姿势类型、镜头距离、自然光、生活感、场景类别、构图节奏。
   - 必须写明 `不照抄参考图模特面貌、发型、表情、穿搭、原场景和姿势`。
   - 背景必须基于参考氛围重新生成，墙面、门窗、路面、家具、绿植、招牌、地面材质、空间布局和构图都要重新设计。

6. 背景不要路人
   - 正向中写明背景干净，画面中只有一个主体模特。
   - 负面中禁止路人、行人、人群、背景人物、游客、骑车人、背景人脸、半截人、虚化人影。

7. 姿势要丰富，不要死板
   - 不要只写直立站着。每次至少给一个自然动作：向前迈步、身体微侧转、整理外套门襟、整理衣领、整理袖口、手扶包带、拿手机、拿饮品、轻靠墙面、站在台阶边、回头但不复刻参考图、自然走动瞬间。
   - 动作不能遮挡核心服装结构，比如胸袋、Logo、门襟、裤腰、裤袋、裤脚。
   - 禁止僵硬正面站姿、双手贴身、动作单一、复刻参考图动作、弯腰系鞋带或低头大幅弯腰，除非用户明确要求。

## 处理流程

1. 识别图片角色
   - 根据用户说明判断：图1上衣/外套，图2内搭，图3裤子，图4场景参考。
   - 如果用户没有明确角色，但图像顺序明显，按商品图在前、参考图在后的方式处理。

2. 提取服装锁点
   - 上衣/外套：颜色、面料、领型、肩部结构、袖长、门襟、纽扣/拉链/搭扣、口袋、拼接线、下摆、版型。
   - 内搭：颜色、领型、袖长、条纹/图案、排扣、层次露出位置。
   - 裤子：颜色、面料、腰头、门襟、皮带袢、口袋、褶裥、裤型、裤脚长度。
   - 配饰：只有用户明确需要时才加入，否则不要让包、帽子、墨镜等抢镜。

3. 生成正向关键词
   - 先绑定服装来源：`使用图1...图2...图3...作为唯一服装参考，服装必须真实还原。`
   - 然后写模特性别、国内语境、全身构图、自然拍摄感。
   - 逐件锁定服装细节。
   - 最后写参考图只借鉴氛围，并重新生成新背景和新动作。

4. 生成负面关键词
   - 禁止照抄参考图人物、脸、发型、穿搭、姿势和场景。
   - 禁止服装颜色、面料、结构、裤型、Logo、口袋等漂移。
   - 禁止半身、裁脚、路人、强滤镜、CG、畸形、模糊、水印。
   - 针对用户提出的失败点，追加精准否定词。

## 常见修补

- 衣服还原错误：加强颜色、面料、版型、领型、袖长、口袋、门襟、Logo/文字和裤型锁定。
- 背景照抄：强调只借鉴氛围，重新设计墙面、门窗、地面、家具、绿植、构图。
- 模特照抄：禁止复制参考图脸型、五官、发型、表情和气质，重新生成新的亚洲模特。
- 姿势太死板：改成走动、转身、整理衣领/袖口/门襟、拿手机/饮品、轻靠墙、台阶边自然停步。
- 背景有路人：正向写单人主体、背景干净；负面加路人、人群、背景人脸、半截人、虚化人影。
- 有滤镜感/棚拍感：正向重写为 `iPhone原相机直出风格、不做后期调色、色彩真实不饱和不过曝、白天户外自然散射光带轻微阴影和轻微过曝、背景适度虚化但保留环境纹理、轻微手机镜头边缘畸变、自然噪点颗粒感、像朋友用手机在街上随手拍的生活照`；负面加棚拍感、商业大片感、专业相机拍摄感、影棚灯光、柔光箱打光、美颜相机感、人像模式过度虚化、HDR合成感。

- 面部瑕疵不适：正向写皮肤干净清爽、自然健康光泽；负面加面部痘印、痘痘、痤疮、明显疤痕、皮肤粗糙、毛孔粗大、油光满面、肤色暗沉不均。不要再用 `真实皮肤带轻微毛孔和瑕疵` 这类表述。
